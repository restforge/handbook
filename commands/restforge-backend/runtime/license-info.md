# `license info`

> Menampilkan informasi license yang aktif pada mesin saat ini (license key, email, type, machine ID, tanggal validasi, dan masa berlaku).

## Pattern

```
npx restforge license info
```

Command ini tidak menerima flag tambahan.

## Contoh

```bat
npx restforge license info
```
