# `license deactivate`

> Melepas aktivasi license dari mesin saat ini sehingga license dapat dipakai pada mesin lain.

## Pattern

```
npx restforge license deactivate
```

Command ini tidak menerima flag tambahan.

## Contoh (Examples)

```bat
npx restforge license deactivate
```
