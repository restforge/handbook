# `project list`

> Menampilkan daftar project yang terdaftar di registry beserta jumlah endpoint, tipe database, dan tanggal pembuatan.

## Pattern

```
npx restforge project list
```

Command ini tidak menerima flag tambahan.

## Contoh

```bat
npx restforge project list
```
