# `config schema`

> Output JSON schema dari parameter `db-connection.env`.

## Pattern

```
npx restforge config schema [--pretty=<BOOL>]
```

## Flag

| Flag | Wajib | Default | Keterangan |
|------|-------|---------|-----------|
| `--pretty <BOOL>` | Tidak | `true` | Pretty-print output JSON |

## Contoh

```bat
npx restforge config schema
```
