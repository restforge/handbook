# `config get-default`

> Menampilkan file config default yang aktif untuk working directory saat ini.

## Pattern

```
npx restforge config get-default [--pretty=<BOOL>]
```

## Flag

| Flag | Wajib | Default | Keterangan |
|------|-------|---------|-----------|
| `--pretty <BOOL>` | Tidak | `true` | Pretty-print output JSON |

## Contoh

```bat
npx restforge config get-default
```
