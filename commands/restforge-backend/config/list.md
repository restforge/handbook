# `config list`

> Menampilkan daftar file `.env` yang tersedia di working directory dan folder `config/`.

## Pattern

```
npx restforge config list [--pretty=<BOOL>]
```

## Flag

| Flag | Wajib | Default | Keterangan |
|------|-------|---------|-----------|
| `--pretty <BOOL>` | Tidak | `true` | Pretty-print output JSON |

## Contoh (Examples)

```bat
npx restforge config list
```
