# `config template`

> Output raw template dari `db-connection.env` (format KEY=VALUE).

## Pattern

```
npx restforge config template
```

Command ini tidak menerima flag tambahan.

## Contoh (Examples)

```bat
npx restforge config template > config/db-connection.env
```
