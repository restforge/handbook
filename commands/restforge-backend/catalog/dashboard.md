# `catalog dashboard`

> Output JSON catalog struktur payload dashboard.

## Pattern

```
npx restforge catalog dashboard [--pretty=<BOOL>]
```

## Flag

| Flag | Wajib | Default | Keterangan |
|------|-------|---------|-----------|
| `--pretty <BOOL>` | Tidak | `true` | Pretty-print output JSON |

## Contoh

```bat
npx restforge catalog dashboard
```
