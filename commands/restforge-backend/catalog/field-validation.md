# `catalog field-validation`

> Output JSON catalog yang berisi tipe field validation, constraint, dan format preset yang didukung.

## Pattern

```
npx restforge catalog field-validation [--pretty=<BOOL>]
```

## Flag

| Flag | Wajib | Default | Keterangan |
|------|-------|---------|-----------|
| `--pretty <BOOL>` | Tidak | `true` | Pretty-print output JSON |

## Contoh

```bat
npx restforge catalog field-validation
```
