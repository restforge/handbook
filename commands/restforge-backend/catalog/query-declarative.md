# `catalog query-declarative`

> Output JSON catalog yang berisi properti query deklaratif, endpoint resolution, dan file reference convention.

## Pattern

```
npx restforge catalog query-declarative [--pretty=<BOOL>]
```

## Flag

| Flag | Wajib | Default | Keterangan |
|------|-------|---------|-----------|
| `--pretty <BOOL>` | Tidak | `true` | Pretty-print output JSON |

## Contoh (Examples)

```bat
npx restforge catalog query-declarative
```
